Модуль ШИФР ЦЕЗАРЯ требует, чтобы вы ввели пятибуквенное слово, сдвигая буквы на дисплее на определенное количество букв вперед или назад. Сдвиг определяется серийным номером реактора. Расшифруйте символы на дисплее с помощью шифра Цезаря. Здесь нет кнопки удаления, так что жмите осторожно!

  2. Последняя цифра серийного номера нечётная +1
  3. Последняя цифра серийного номера чётная -1
  4. Количество символов сообщения нечётное +1

Модуль Незабудка – это секретный модуль. Он оснащен простой клавиатурой 0-9 и двумя дисплеями, на одном из которых отображается номер этапа, на другом – случайное число. Ваша задача добавить отображаемое случайное число к другому числу, вычисленному по другим признакам, приведённым в подсказке.

  1. Елси третья цифра серийного номера реактора чётная, то +5, -3, -1
  2. Если третья цифра серийного номера реактора нечётная, то +3, -1, -7
  3. Если весь серийный номер реактора чётное число, то -3, +1, +7

Модуль цветной вспышки будет многократно выдавать последовательность из 8 различных слов, представляющих цвета в разных цветах. Возможные цвета: красный, желтый, зеленый, синий, фиолетовый и белый. На модуле также есть кнопки "Да" и "Нет". Для отключения модуля необходимо нажать только одну из кнопок "Да" и "Нет", но она должна быть нажата в нужное время в соответствии с приведенными ниже правилами.
